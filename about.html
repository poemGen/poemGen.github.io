About Nelson Library's Poetry Generator

The poemGen poetry generator is a project initiated by the Nelson Public Library for the 2016 New Zealand Poetry Month (August). The idea: an oldschool terminal (reusing e-waste) running a retro styled application to generate passable poetry.

At the heart of the project is a modified version of Zack (schollz) poetry generator (https://github.com/schollz/poetry-generator) with an extended and corrected .YAML file. The generator is then run through a custom pygame wrapper that simulates a MS-DOS terminal.

The system will be running in the Nelson Library from th 15th of August and selected output will be showcased here.
