About Nelson Library's Poetry Generator

The poemGen poetry generator is a project initiated by the Nelson Public Library for the 2016 New Zealand Poetry Month. The idea: an oldschool terminal (using e-waste) running a retro styled application to generate passable poetry.

The project is based on a modified version of Zack's (schollz) poetry generator (https://github.com/schollz/poetry-generator) with an extended and corrected .YAML file. Then run through a custom pygame wrapper that simulates a MS-DOS terminal.

The system will be running for two weeks beginning 15th August and selected output will be showcased here.
